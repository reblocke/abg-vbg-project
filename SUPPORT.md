# Support

## Getting help
- **Questions / usage:** open a GitHub Issue with the label `question`
- **Bugs:** open an Issue with a minimal reproducible example
- **Reproducibility problems:** include:
  - OS + R version
  - the exact command(s) run
  - session info (`sessionInfo()`)

## What to include in an issue
- What you expected to happen
- What actually happened
- Steps to reproduce
- Relevant logs / error messages

## Response times
This is a research codebase and support may be best-effort. If you need a response for a deadline,
flag it clearly in the issue title.
